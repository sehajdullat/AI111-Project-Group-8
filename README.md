# 🚦 AI Traffic Optimization with Emergency Handling

## Overview
Multi-file modular implementation of traffic signal optimization using Poisson distribution, queueing theory, and emergency prioritization.

## Run
pip install -r requirements.txt
python main.py

## Files
config.py → parameters  
utils.py → math & generation  
simulation.py → logic  
display.py → output  
main.py → execution  

## Innovation
Score = Q + λT (predictive) + emergency override
